Tipos de selectores que entran en examen

Clase

Ejemplo de elemento con clase.

El selector de clase selecciona elementos que tienen asignada la clase especificada. Es útil para aplicar estilos a varios elementos de forma consistente.

Identificador (#identificador)

Ejemplo de elemento con identificador.

El selector de identificador selecciona un elemento específico que tiene el identificador especificado. Debe ser único en la página y es útil para aplicar estilos o manipular ese elemento en particular.

Todos los elementos (*)

Este selector selecciona todos los elementos en la página.

Es útil para aplicar estilos generales a todos los elementos de una página web.

Varios elementos (p, em, h1, h2)

Selecciona varios tipos de elementos específicos para aplicar estilos de manera conjunta o realizar manipulaciones.

Sobrevuelo (p:hover)

Sitúa el cursor sobre este párrafo para ver el efecto de hover.

El selector :hover permite aplicar estilos cuando el usuario coloca el ratón sobre un elemento. Es útil para crear interactividad y feedback visual en la interfaz de usuario.
